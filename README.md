# zimlet-blueprint
A blueprint for zimlet (zimbra 8) plugin development.

It shows how to:
* Add a toolbar button
* Read content of the view
* Localization
* Use custom css
* Use icons

## "Bundling"

Create a zip with the same file name as your xml.
